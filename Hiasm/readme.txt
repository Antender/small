Программа для удерживания на виду нужных окон
Автор: Волокитин А.В.
Лицензия: сокращённая BSD

Перемещает от курсора окна заданные в masks.cfg

Левый клик по иконке в трее - временно отключает/включает программу
Правый клик  - вызвать меню


Состав config.ini:

CheckDelay= время проверки окон
CheckGeometry= 
	False - проверяет попадание в окно по идентификатору
	True - по размерам окнам
